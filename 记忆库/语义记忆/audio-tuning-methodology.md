# 语义记忆：音频调教方法论

## V2 引擎质感公式

### 寒意核心参数
```python
STABILITY = 0.22        # 极低稳定性产生寒意起伏
SIMILARITY = 1.0        # 满级相似度死磕声纹细节
STYLE = 0.9             # 高风格化强化个性表达
```

### 底层逻辑

#### 1. Stability（稳定性）
- **范围**: 0.0 - 1.0
- **冷酷军师配置**: 0.22
- **原理**: 极低稳定性让声音产生自然波动和情绪起伏
- **效果**: 打破机械朗读感，注入"人性寒意"

#### 2. Similarity Boost（相似度增强）
- **范围**: 0.0 - 1.0  
- **冷酷军师配置**: 1.0（满级）
- **原理**: 强制锁死声纹特征，确保音色一致性
- **效果**: 声音厚度和磁性达到最大值

#### 3. Style（风格化）
- **范围**: 0.0 - 1.0
- **冷酷军师配置**: 0.9
- **原理**: 高风格化放大声音的个性和表现力
- **效果**: 强化"不屑"和"专业"的气质

### 脱洋逻辑注入

**物理停顿替换**：
```python
script = (
    raw_script
    .replace("。", "... ...")  # 句号 → 长停顿
    .replace("！", "... ...")  # 感叹号 → 长停顿
    .replace("？", "... ...")  # 问号 → 长停顿
)
```

**作用**：
- 强迫 TTS 引擎在狠话后执行物理吸气
- 粉碎英文腔的连续朗读节奏
- 注入中式演讲的"呼吸感"

### 质感公式总结

```
冷酷军师音频 = 
  极低稳定性（0.22）寒意起伏 
  + 满级相似度（1.0）声纹锁死 
  + 高风格化（0.9）个性放大
  + 物理停顿（... ...）中式吸气
```

### 实战验证
- ✅ 消除朗读感
- ✅ 注入人性寒意
- ✅ 粉碎洋腔洋调
- ✅ 40秒脚本最佳适配

### 扩展可能
- 不同行业可调整 STYLE（0.7-0.9）
- 女性声音可微调 STABILITY（0.25-0.30）
- 但 SIMILARITY 必须保持 1.0（质感核心）
